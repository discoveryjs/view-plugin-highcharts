# @discoveryjs/view-plugin-highcharts

Highcharts plugin for Discovery.js

## Usage

```
npm i @discoveryjs/view-plugin-highcharts
```

Use in discovery.js config:

```js
{
    ...
    plugins: [
        '@discoveryjs/view-plugin-highcharts',
        '@discoveryjs/view-plugin-highcharts/index.css'
    ]
}
```
